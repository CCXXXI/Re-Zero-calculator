[交互模式]
交互模式 = 图形界面

# 交互模式默认为图形界面(GUI)，在此模式下，输入输出都会通过图形界面进行

# 其他可选交互模式包括：命令行、文件
# 命令行：通过命令行指定魔法器词条数等选择，在命令行中输出结果
# 文件：通过本文件指定各种选择，在命令行中输出结果

# 质量系数为0.8，表示词条属性按最大值的80%计算(大概是B级词条)
# 除了血防和伤害减免以外的词条，视作有效词条，最多12条，不算固定词条
